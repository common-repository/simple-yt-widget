== Description ==

Adds customizable YouTube widget to your website. No coding required. Just drag-and-drop and you're good to go.

Displays a customizable and simple YouTube Channel Widget for your website.

== Installation ==

Activate the plugin through the *Plugins* menu in wordpress.

Or upload the zip file through the *Add New* submenu of the *Plugins* menu in wordpress.

== How to use? ==

1. Activate the plugin.
2. Go to *Widgets* submenu in the *Appearance* section of WordPress.
3. Drag the *Simple YT Widget* to your specified widget area.
4. Customize the title and channel id of your widget and you're all set.

Your YouTube Channel ID can be found [here](https://support.google.com/youtube/answer/3250431).
